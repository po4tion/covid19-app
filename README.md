## About covid19-app(Covid Data)

- 우리나라 COVID-19 확진자, 사망자 등의 정보 제공
- 세계 COVID-19 확진자, 사망자 정보 제공
- 시각적 효과를 극대화하기 위한 라인 차트 제공
- 사용자의 눈의 피로를 줄이기 위한 다크모드 제공
- 국내 지역별, 세계 나라별 확잔지/사망자 정보 제공
